# `clear()`
Clear text.

- **type**: Method
- **return type**: Void
- **availability**: `iOS 8+`, `tvOS 9+`.
- **unit tests**: `iOS`, `tvOS`.

Example

```swift
let textView = UITextView()
textView.clear()
```
