## `Operator *`
Repeat character multiple times.

 - **type**: Infix Operator.
 - **return type**: String
 - **returns**: string with character repeated n times.
 - **availability**: `iOS 8+` `tvOS 9+` `watchOS 3+` `macOS 10.10+`.

Example

```swift
'-' * 10 -> "----------"
15 * 'a' -> "aaaaaaaaaaaaaaa"
```
